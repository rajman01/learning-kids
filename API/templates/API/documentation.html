Documentation


Registering a User

Request: POST

Url:

https://learning-kids.herokuapp.com/api/register/

Params:

username (Your username)

email (Your email)

password1 (first password)

password2 (confirm password)


Success Response:

{
    "response": "successfully registered a Parent account",
     "email": "lekan@gmail.com",
    "username": "lekan",,
     "token": "ef9153a6f4d842cca47808f0c16fde814b2ff9fb"
}

NB: A default parent profile is created for a newly registered user

Obtaining an Authentication Token

Request: POST

Url:

https://learning-kids.herokuapp.com/api/login/

Params:

username (Your username)

password (Your password)


Success Response:

{
    "response": "Successfully authenticated.",
    "pk": 9,
    "username": "lekan",
     "token": "ef9153a6f4d842cca47808f0c16fde814b2ff9fb"
}

Viewing a Specific user

Request: GET

Headers:

Authorization: Token <token>

Url:

https://learning-kids.herokuapp.com/api/user/<pk>/

Success Response:

{
    "url": "http://127.0.0.1:8000/api/user/5",
