Documentation


Registering a User

Request: POST

Url:

https://learning-kids.herokuapp.com/api/register/

Params:

username (Your username)

email (Your email)

password1 (first password)

password2 (confirm password)


Success Response:

{
    "response": "successfully registered a Parent account",
     "email": "lekan@gmail.com",
    "username": "lekan",,
     "token": "ef9153a6f4d842cca47808f0c16fde814b2ff9fb"
}

NB: A default parent profile is created for a newly registered user

Creating a child Profile or register a new child

Request: POST

Headers:

Authorization: Token <token>

Url:

https://learning-kids.herokuapp.com/api/register_child/

Params:

parent_username (username)

child_name (child name must be unique for each parent)

age (child age)

gender (child gender)


Success Response:

{
    "response": "successfully created",
    "name": "Lekan",
    "age": 1,
    "gender": "Male"
}

List categories (Pagination, Search and Ordering)

Request: GET

Headers:

Authorization: Token <token>

Url:

https://learning-kids.herokuapp.com/api/categories/?search=<query>&ordering=-id&page=1

Params:

query (this will search the name of the category for a match)

ordering (Can use: id, -id, name, -name)

page (15 results will be shown per page.)
